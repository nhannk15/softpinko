Softy Pinko Landing Page

A static landing page built with HTML, CSS, and vanilla JavaScript using Bootstrap 4. Practice project from the 28Tech Front-End course.

## Features

- Responsive layout with Bootstrap 4
- Scroll animation using IntersectionObserver
- Header background changes on scroll
- Sections: Home, About, Work Process, Testimonials, Pricing Plans

## Getting Started

This is a static HTML project — **no installation or build step required**.

### Option 1: Open directly

Open `index.html` in any browser (Chrome, Firefox, Edge, etc.).

### Option 2: Live Server (recommended)

In VS Code, install the **Live Server** extension, then right-click `index.html` > **Open with Live Server**.

### Option 3: http-server (Node.js)

```bash
npx http-server .
```

Then visit `http://localhost:8080`.
